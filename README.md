# n9wu_sidequest_W4

This demo shows a small platformer that loads level data from JSON and uses classes.

How to run

- Open [index.html](index.html) in a browser, or serve the folder and visit http://localhost:8000

Quick local server (Python):

```bash
python -m http.server 8000
```

Controls: Arrow keys to move, `Space` or `ArrowUp` to jump. Complete level 1 to auto-load level 2.
